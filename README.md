# **Project Name**

This repository contains the implementation of **OBFNet: A Neural Beamforming Network Using Orthonormal Basis Functions**.

------------------------------------------------------------------------

## **Introduction**

This project implements **OBFNet**,
designed for **multichannel speech enhancement / neural beamforming**.

The framework is developed using **PyTorch**,
and supports training and evaluation on datasets such as **CHiME-4 / VOiCES / RealMAN**.
